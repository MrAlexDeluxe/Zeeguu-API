Zeeguu Installation Instructions


For The Chrome Browser

From your Chrome Browser, go to the Chrome Extension Store and install the Zeeguu extension.
Or point your browser to:
  • http://mir.lu/z/chrome
After the installation remember that:
  1. Before using the plugin you must click on the (newly available) elephant icon in your toolbar and login with your Zeeguu account
  2. You must enable the plugin on every website by clicking the grayed out elephant icon and pressing the on/off button from the dropdown menu.
  3. To get translations for words you must hold the alt key and select the text you want to translate. To automatically translate selected text, you must select the little lightning button from the same dropdown menu of earlier.

For Android

Because the app is in Alpha version, it is not publicly available on the Google Play Store.
To install it you must follow the next two steps:

  1. Join the The Zeeguu Alpha Testers Google+ Community by following one of the two links:
    • https://plus.google.com/communities/107424205336144958718
    • http://mir.lu/z/plus
  2. From your phone go to one of the following two links to install the app:
    • https://play.google.com/apps/testing/ch.unibe.scg.zeeguu
    • http://mir.lu/z/android
